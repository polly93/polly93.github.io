은행 지점 검색 관심도 분석

은행 지점 축소와 디지털 전환 과정에서 소비자의 오프라인 지점 관심도를 API 데이터로 측정한 프로젝트입니다.

문제 정의

디지털 뱅킹 확산은 오프라인 지점에 대한 관심을 줄였는가?

은행 지점 수가 감소하는 과정에서 개별 지점에 대한 소비자 검색 관심도가 어떻게 변화했는지 분석합니다.

데이터 및 분석 과정

분석 가능한 데이터 구축

  • 6,000개 이상의 지점별 검색 키워드를 생성했습니다.
  • API 결과를 수집·가공해 은행×시점 패널을 구축했습니다.
  • 지점 위치와 행정구역을 지리정보 도구로 검증했습니다.

분석 방법

패널 구축, 추세 분석, 탐색적 모형

은행, 지점, 시점별 검색 패턴을 비교하고 데이터 수집 및 변환 절차를 문서화했습니다.

해석

이 분석이 보여주는 역량

API 활용, 비정형 데이터 정리, 패널 데이터 설계, 소비자 디지털 행동의 비즈니스 지표화를 보여줍니다.
