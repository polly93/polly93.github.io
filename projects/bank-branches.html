BANKING · API DATA PIPELINE

Bank Branch Search Trends

An API-based panel-data project measuring changes in consumer interest as bank branch networks contracted.

PROBLEM

Did digital banking reduce interest in physical branches?

The project examines whether consumer search interest around individual bank branches declined alongside branch closures and banking’s broader digital transition.

DATA & WORKFLOW

Building an analysis-ready dataset

  • Generated more than 6,000 branch-specific search keywords.
  • Collected and transformed API output into a bank-by-time panel.
  • Validated branch locations and administrative areas using geographic tools.

METHOD

Panel construction, trend analysis, and exploratory modeling

Compared search patterns across banks, branches, and time while documenting the data collection and transformation workflow.

INTERPRETATION

What the analysis contributes

The case study demonstrates practical API use, messy-data transformation, panel-data thinking, and the translation of digital behavior into a business-relevant measure.
