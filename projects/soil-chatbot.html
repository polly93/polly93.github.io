MACHINE LEARNING · CLASSIFICATION

Land-Use Recommendation Chatbot

A neural-network classification project that converts soil characteristics into land-use recommendations.

PROBLEM

Can complex soil records support a simple recommendation interface?

The team developed a prototype that recommends an appropriate land-use category from observed soil characteristics.

DATA & WORKFLOW

Building an analysis-ready dataset

  • Cleaned and transformed more than 100,000 soil records.
  • Prepared labels and model-ready features.
  • Collaborated on integrating model predictions into a chatbot prototype.

METHOD

Neural-network classification

Trained and evaluated a Keras deep-neural-network classifier, with attention to preprocessing, feature representation, and out-of-sample performance.

INTERPRETATION

What the analysis contributes

The project demonstrates how a predictive model can be embedded in a user-facing workflow rather than remaining a standalone notebook.
